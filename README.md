# knock-knock

CLI tool for obtaining domain name information


## How to use

```console
knock-knock \
    duckduckgo.com \
    github.com
```

#### Sample output

```
duckduckgo.com:
    Domain name will expire in 271 days, 19 hours, 25 minutes, 22 seconds
github.com:
    Domain name will expire in 380 days, 9 hours, 46 minutes, 13 seconds

```
